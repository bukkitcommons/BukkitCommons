About
---
> The standard Bukkit libraries fail to provide enough methods for manipulation of its core classes.

Bukkit Commons is a next-generation plugin development framework for [Bukkit API](https://dev.bukkit.org/).

### Build
#### Requirements
* Java (JDK) 8 or above
* Maven

#### Compile
```sh
mvn clean install
```
